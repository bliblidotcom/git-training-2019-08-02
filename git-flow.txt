1. Buat branch baru untuk setiap fitur yang akan dibuat dan bug yang di-resolve, dapat juga hanya 1 branch sesuai dengan kebutuhan. Lakukan perintah berikut untuk membuat branch baru :
   git checkout -b 'new-branch-name'
2. Setelah berada di branch, lakukan perubahan sesuai dengan permintaan, dapat berupa penambahan fitur atau penyelesaian bug.
3. Setelah menambahkan fitur, lakukan perintah berikut untuk melakukan perubahan pada remote branch dari local branch :
   git add .    // menambahkan semua file yang berada di file repository lokal
   git commit -m 'commit message'   // melakukan commit dilengkapi dengan pesan perubahan
   git push origin 'new-branch-name'    // memasukkan perubahan pada remote branch
4. Setelah menyelesaikan masing-masing fitur dan bug, pindah ke branch master untuk melakukan merging dengan perintah berikut.
   git checkout master
5. Pada saat di branch master, lakukan merging terhadap branch lainnya dengan perintah berikut.
   git merge 'new-branch-name'
6. Jika terdapat konflik, tangani dulu konflik yang terjadi sebelum melakukan perubahan pada remote master.
7. Untuk melakukan perubahan pada remote master, lakukan perintah berikut.
   git add .
   git commit -m 'commit message'
   git push origin master
8. Selama proses pengerjaan, dapat digunakan git log dan git status untuk membantu pengerjaan.
